# 💬 Message Box Character Counter

A simple **Message Box with Character Counter** built using **HTML, Tailwind CSS, and JavaScript**.
This project allows users to type messages while tracking the number of characters entered and displaying a warning when the limit is exceeded.

---

# 🚀 Features

* Responsive UI using **Tailwind CSS**
* **Live character counter**
* **Maximum character limit**
* **Warning message when the limit is exceeded**
* Simple and clean interface
* Real-time input tracking using **JavaScript**

---

# 🛠 Technologies Used

* **HTML5** – Structure of the message box
* **Tailwind CSS** – Styling and layout
* **JavaScript (DOM Manipulation)** – Character counting and validation

# 💡 Functionality

## Character Counter

* Displays the **current number of characters typed**
* Updates automatically when the user types

Example:

```id="code6"
120 / 200
```

---

## Character Limit

* Maximum allowed characters: **200**
* If the user exceeds the limit:

  * Extra characters are prevented
  * A **warning message** is displayed

Example warning:

```id="code7"
Maximum character limit reached
```
